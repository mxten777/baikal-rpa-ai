## 솔루션 소개

### 🏢 BAIKAL RPA AI란?

**BAIKAL RPA AI**는 기업 내부 업무를 AI(인공지능)와 RPA(로봇 프로세스 자동화)로 혁신하는 **올인원 업무 자동화 플랫폼**입니다.

사내 전용 ChatGPT 같은 AI 어시스턴트, AI 기반 문서 자동 생성, 웹 데이터 수집 및 엑셀 자동 처리까지 — 반복적이고 시간이 많이 걸리는 업무를 자동화하여 **임직원의 생산성을 비약적으로 향상**시킵니다.

## 핵심 기능

### 1. 🤖 AI 어시스턴트 (사내 전용 AI 챗봇)

> **"물어보세요. AI가 바로 답해드립니다."**

- ChatGPT 수준의 고성능 AI를 사내 전용으로 운영
- 업무 질의응답, 아이디어 브레인스토밍, 데이터 분석 지원
- 대화 맥락을 이해하는 연속 대화 지원
- OpenAI GPT-4 또는 사내 로컬 AI(Ollama) 선택 가능

**활용 예시:**
- "이번 분기 마케팅 전략을 5가지로 제안해줘"
- "이 회의록 내용을 3줄로 요약해줘"  
- "고객 불만 대응 이메일 템플릿을 만들어줘"

---

### 2. 📄 AI 문서 자동 생성

> **"제목과 핵심 내용만 입력하면, AI가 완성된 문서를 작성합니다."**

- **보고서**: 업무 보고서, 프로젝트 보고서, 실적 분석 보고서
- **공문**: 협조 요청 공문, 회의 안내문, 인사 공문
- **이메일**: 거래처 안내 메일, 미팅 요청 메일, 감사 메일

**특징:**
- 문서 유형에 최적화된 전문 AI 프롬프트
- 한국어 비즈니스 문서 형식 준수
- 생성 즉시 저장 및 관리
- 원클릭 미리보기

---

### 3. 🌐 웹 스크래핑 자동화

> **"웹사이트 데이터를 24시간 자동으로 수집합니다."**

- 경쟁사 가격 모니터링
- 뉴스/기사 자동 수집
- 공공 데이터 정기 수집
- 웹 테이블 데이터 자동 추출

**특징:**
- Playwright 기반 최신 브라우저 엔진
- JavaScript 렌더링 웹페이지도 지원
- CSS 선택자로 정확한 데이터 타겟팅
- 텍스트 / HTML / 테이블 다양한 추출 모드

---

### 4. 📊 엑셀 자동 처리

> **"엑셀 데이터 정리, 클릭 한 번으로 끝냅니다."**

- 빈 행/중복 데이터 자동 제거
- 데이터 자동 정렬
- 기초 통계 자동 분석 (평균, 최솟값, 최댓값 등)
- 처리 결과 파일 자동 저장

---

### 5. ⏰ 스케줄 자동 실행

> **"설정만 해두면, 정해진 시간에 알아서 실행됩니다."**

- Cron 기반 유연한 스케줄 설정
- 평일 오전 9시 실행, 매시간 실행, 매월 1일 실행 등
- 실행 결과 자동 기록 및 로그

---

### 6. 📊 대시보드

> **"업무 현황을 한눈에 파악합니다."**

- AI 대화 횟수, 문서 생성 수, 자동화 실행 현황
- 시스템 상태 실시간 모니터링
- 직관적인 통계 카드 UI

## 기술 아키텍처

### 검증된 기업급 기술 스택

```
┌──────────────────────────────────────────────┐
│            사용자 인터페이스                     │
│     React 18 + TailwindCSS (반응형 웹)         │
├──────────────────────────────────────────────┤
│              API 서버                          │
│     FastAPI (비동기 고성능 Python)               │
├─────────────┬──────────────┬─────────────────┤
│   AI 엔진   │   RPA 엔진   │    스케줄러      │
│ OpenAI/Ollama│ Playwright   │   Cron 기반     │
│ GPT-4/LLaMA │ Pandas       │   정기 실행     │
├─────────────┴──────────────┴─────────────────┤
│              데이터베이스                       │
│     PostgreSQL (운영) / SQLite (개발)          │
└──────────────────────────────────────────────┘
```

### 기술 하이라이트

| 항목 | 적용 기술 | 장점 |
|------|-----------|------|
| **AI 엔진** | OpenAI GPT + Ollama | 클라우드/로컬 AI 선택 가능 |
| **API 서버** | FastAPI (async) | 높은 동시 처리 성능 |
| **RPA 엔진** | Playwright + Pandas | 최신 웹 브라우저 자동화 |
| **데이터베이스** | PostgreSQL + SQLAlchemy | 안정적인 데이터 관리 |
| **프론트엔드** | React + TailwindCSS | 모바일 반응형 UI |
| **인증 보안** | JWT + bcrypt | 업계 표준 보안 |
| **배포** | Docker Compose | 원클릭 배포 |

## 보안 및 컴플라이언스

### 🔒 기업 데이터 보안

| 보안 항목 | 적용 현황 |
|-----------|-----------|
| **비밀번호 암호화** | bcrypt 해시 (업계 최고 표준) |
| **API 인증** | JWT Bearer Token |
| **데이터 격리** | 사용자별 완전 데이터 분리 |
| **통신 보안** | HTTPS (TLS 1.3) 지원 |
| **CORS 제어** | 화이트리스트 기반 접근 통제 |
| **로컬 AI 옵션** | Ollama 사용 시 데이터 외부 미전송 |

### 🏠 온프레미스 지원

민감한 기업 데이터를 외부로 전송하지 않도록, **완전한 온프레미스 배포**를 지원합니다:
- Ollama 로컬 AI → 사내 서버에서 AI 엔진 직접 구동
- Docker 기반 전체 스택 → 사내 인프라에 완전 독립 운영
- 외부 클라우드 의존성 제로 (OpenAI API 미사용 시)

```
     ① 사전 미팅          ② PoC (2주)          ③ 커스터마이징         ④ 운영 전환
   ┌──────────┐      ┌──────────────┐      ┌──────────────┐      ┌──────────┐
   │ 요구사항  │ ──▶  │ 테스트 환경  │ ──▶  │ 맞춤 개발    │ ──▶  │ 정식 운영 │

```

### 단계별 상세

| 단계 | 기간 | 내용 |
|------|------|------|
| **① 사전 미팅** | 1~2일 | 고객 업무 프로세스 분석, 자동화 대상 선정, 요구사항 정의 |
| **② PoC** | 1~2주 | 테스트 환경 구축, 핵심 시나리오 검증, 성능 확인 |
| **③ 커스터마이징** | 2~4주 | UI 커스텀, 기존 시스템 연동, 업무별 자동화 시나리오 개발 |
